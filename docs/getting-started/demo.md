## 화면 구성

### 메인 뷰

- **중앙 캔버스** — 울산항 3D 미니어처 디지털트윈
- **상단 좌측** — 핵심 운영 지표 (선석 점유율 / 야드 평균 / CO2 / 알림 / 시뮬레이션 시간)
- **상단 우측** — 객체 정보 패널 (선택된 객체의 상세 정보 + 관련 객체 + 검색 + 데이터 임포트)
- **상단 좌측 (지표 아래)** — 컨트롤 패널 (오버레이 / 카메라 프리셋 / 표시 옵션)
- **하단 중앙** — 타임라인 패널 (재생/일시정지, 시간 스크러버, 속도)
- **하단 우측** — 필터 패널 (엔티티 종류 / 관계 타입 토글)

### Knowledge Graph 뷰

컨트롤 패널의 **Graph** 토글(또는 `G` 키)로 전체 화면 노드-엣지 그래프 뷰 전환.
온톨로지 관계가 강조되며, 노드를 클릭하면 메인 뷰의 선택 상태가 동기화됩니다.

## 인터랙션 가이드

### 마우스 / 터치

| 동작 | 효과 |
|---|---|
| 객체 클릭 / 탭 | 해당 객체 선택 + 관계 라인 시각화 + 상세 패널 표시 |
| 빈 공간 클릭 | 선택 해제 |
| 마우스 드래그 / 한 손가락 드래그 | 카메라 회전 |
| 스크롤 / 두 손가락 핀치 | 줌 인/아웃 |
| Shift + 드래그 / 두 손가락 드래그 | 카메라 이동 |
| 객체 호버 | 라벨 표시 |

### 키보드 단축키

| 키 | 동작 |
|---|---|
| `1` / `2` / `3` / `4` | 오버레이 모드 (Default / Congestion / Delay / Carbon) |
| `R` / `T` / `Y` | 카메라 프리셋 (Reset / Top / Side) |
| `G` / `L` / `E` | Graph / Labels / Relations 토글 |
| `F` | Time filter ON/OFF |
| `Space` | 타임라인 재생 / 일시정지 (time filter ON 상태에서) |
| `/` | 검색창 포커스 |
| `Esc` | 선택 해제 |

## 추천 탐색 경로

1. **항만 전체 보기** — 메인 뷰에서 울산항 전체 구조 확인
2. **검색** — `/` → "HMM" 입력 → Enter로 즉시 이동
3. **오버레이 전환** — 컨트롤 패널의 Congestion / Delay / Carbon 모드 차례로 시도 (`2` → `3` → `4`)
4. **선박 선택** — 대기 중인 `HMM Promise` 클릭 → 지연 이벤트 및 영향 객체 확인
5. **타임라인 재생** — `F`로 시간 필터 켜고 `Space`로 재생 → 이벤트가 시간순으로 등장
6. **필터링** — 우측 하단 Filters에서 Yard만 끄고 보기
7. **공유** — 선택된 상태에서 URL 복사 → `?selected=vessel_007` 자동 deep-link

## 커스텀 데이터 로드

기본 데모 외에 자체 데이터셋을 시각화할 수 있습니다:

1. 정보 패널의 **⬆ Load JSON** 버튼 클릭
2. `PortData` 스키마 (port, terminals, berths, yardBlocks, gates, vessels, containers, events, emissions, relations)를 만족하는 JSON 업로드
3. 잘못된 스키마면 검증 오류 메시지 표시
4. **Reset** 버튼으로 기본 데모 복귀

스키마 자세한 내용은 [데이터 모델](../domain/data-model.md) 참고.
