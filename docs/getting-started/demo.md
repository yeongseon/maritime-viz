## 화면 구성

### 메인 뷰

- **중앙 캔버스** — 부산항 3D 미니어처 디지털트윈
- **상단 좌측** — 핵심 운영 지표 (선석 점유율 / 야드 평균 / CO2 / 알림)
- **상단 우측** — 객체 정보 패널 (선택된 객체의 상세 정보 + 관련 객체)
- **하단 좌측** — 컨트롤 패널 (오버레이 모드 + 표시 옵션)

### Knowledge Graph 뷰

컨트롤 패널에서 **Graph** 토글을 켜면 전체 화면 노드-엣지 그래프 뷰로 전환됩니다.
온톨로지 관계가 강조되며, 노드를 클릭하면 메인 뷰의 선택 상태가 동기화됩니다.

## 인터랙션 가이드

| 동작 | 효과 |
|---|---|
| 객체 클릭 | 해당 객체 선택 + 관계 라인 시각화 + 상세 패널 표시 |
| 빈 공간 클릭 | 선택 해제 |
| 마우스 드래그 | 카메라 회전 |
| 스크롤 | 줌 인/아웃 |
| Shift + 드래그 | 카메라 이동 |
| 객체 호버 | 라벨 표시 |

## 추천 탐색 경로

1. **항만 전체 보기** — 메인 뷰에서 부산항 전체 구조 확인
2. **오버레이 전환** — 컨트롤 패널의 Congestion / Delay / Carbon 모드 차례로 시도
3. **선박 선택** — 대기 중인 `HMM Promise` 클릭 → 지연 이벤트 및 영향 객체 확인
4. **터미널 선택** — `Sinseondae Terminal` 클릭 → 야드 적치율과 게이트 대기열 확인
5. **Knowledge Graph** — Graph 모드로 전체 관계 구조 탐색
