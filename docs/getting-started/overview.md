# 프로젝트 개요

## 프로젝트명

**해운물류 온톨로지 기반 3D 시각화 플랫폼**

## 목표

해운물류 도메인의 핵심 객체, 관계, 이벤트를 **온톨로지 기반**으로 구조화하고, 이를 **3D 공간에서 탐색 가능**하도록 시각화하여 다음을 가능하게 합니다.

- 해운물류 흐름의 구조적 이해
- 병목 구간의 시각적 식별
- 지연 원인의 관계 기반 탐색
- 항만 운영 상태의 직관적 파악
- 탄소 및 운영효율 이슈의 동시 확인

## 차별점

본 프로젝트는 다음과 같은 기존 솔루션과 명확히 구분됩니다.

| 구분 | 본 프로젝트 | 기존 지도 서비스 | 기존 대시보드 | 단순 3D 모델링 |
|---|:---:|:---:|:---:|:---:|
| 위치 정보 | ✅ | ✅ | ⚠️ | ✅ |
| 통계/지표 | ✅ | ❌ | ✅ | ❌ |
| **온톨로지 관계** | ✅ | ❌ | ❌ | ❌ |
| **의미 기반 3D** | ✅ | ❌ | ❌ | ⚠️ |
| 운영 이슈 통합 | ✅ | ❌ | ⚠️ | ❌ |

> 본 프로젝트는 **단순 지도/대시보드/3D 모델링이 아니라, 온톨로지 기반 의미 관계를 반영한 3D 시각화 플랫폼**입니다.

## 기대 효과

- 복잡한 물류 관계를 **누구나 이해하기 쉬운 형태**로 제공
- 2D 지도와 표 중심 화면보다 **높은 설명력 확보**
- 공모전/심사 시 **시각적 임팩트와 기술적 차별성** 동시 확보
- 향후 **디지털트윈, 물류 분석, 의사결정 지원 시스템**으로 확장 가능

## MVP 범위

본 프로젝트는 MVP 단계에서 글로벌 전체 물류망을 모두 다루지 않고, **부산항 1곳을 중심으로 한 해운물류 흐름 시각화**를 우선 대상으로 설정합니다.

### 포함

- 부산항 3개 터미널, 8개 선석, 9개 야드 블록, 3개 게이트
- 8척의 선박 (입항/접안/대기/출항 상태)
- 컨테이너 이동 흐름 표현
- 혼잡, 지연, 체류, 탄소 관련 상태 표시
- 온톨로지 기반 관계 그래프 뷰

### 제외 (향후 확장)

- 전 세계 항만 실시간 연동
- 자율운항 제어 기능
- 초대형 시뮬레이션 엔진
- 실제 운영 시스템과 완전 통합된 디지털트윈
